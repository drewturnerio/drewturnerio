# Contributing to this project

Thank you for your interest in my cybersecurity learning journey. 

This repository serves as a personal documentation hub for my lab projects, technical notes, and certification preparation. While this is primarily a personal reference, I value collaboration and the spirit of open-source knowledge sharing.

## How to Contribute
If you are an industry peer, student, or mentor and would like to help improve this documentation:

### 1. Reporting Issues
If you encounter a mistake in a lab configuration, broken links, or inaccurate technical information, please open an **Issue** in this repository. Please include:
* A clear description of the issue.
* The specific lab or file path where it occurs.
* Any suggested corrections.

### 2. Suggesting Improvements
If you have suggestions for better security practices or more efficient ways to structure my documentation:
* Fork this repository.
* Create a new branch for your changes.
* Submit a **Pull Request** detailing the improvements.

### 3. General Feedback
If you have questions about my setup or would like to discuss my findings, feel free to connect with me via [drewturner.io](https://drewturner.io).
